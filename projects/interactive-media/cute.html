LINK TO WEBSITE: CUTE!

Technologies: p5.js, Ableton Live

Type: Interactive Sketch

Cute! is an interactive sketch that invites users to grow, pop and create bubbles in a soft pastel world. Each bubble reacts to user input, growing when pressed, fading when collided and triggering musical notes that was composed in Ableton Live. The project explores the idea of cuteness through motion, sound and responsiveness, aiming to evoke care and playfulness in simple interactions.

KEY SKILLS

  • p5.js and object oriented programming
  • Interactive sound design and synthesis
  • Collision detection and visual feedback systems
  • State based user interaction

The code uses class-based structures to manage bubble behavior, including spawning, collision detection, growth and fading. A color-lerped background gives the canvas a gentle sense of life, while each bubble has a unique phase offset to create rhythmic but unsynchronized pulsing. Sounds are mapped and all kept within the key of C for warmth and familiarity.
