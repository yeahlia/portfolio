LINK TO WEBSITE: Chaos

Technologies: p5.js, Web Audio API

Type: Generative Interactive Artwork

Chaos is an evolving generative artwork that was inspired by Merlin Sheldrake's "What Is It Like to Be a Fungus?" and the concept of effective complexity to balance order and chaos. It simulates an organic branching system that continuously grow, decay and regrow using recursive structures and random chance to evoke a living, breathing ecosystem.

KEY SKILLS

  • Recursive generative systems and effective complexity
  • p5.js creative coding
  • Interactive sound design with web audio API
  • Algorithmic animation and real time interactivity
  • Conceptual translation of ecological growth and decay

Each branch behaves autonomously yet remains connected to its parent, which creates a network that endless shifts between growth and collapse. Over time, user interactions such as mouse movements or clicks can spawn new colonies, each with different colors, sound oscillations and physics-influenced swaying motion. There are also audio-reactive oscillators and vibrato effects that link the visual and sound layers to form a living, fungal-like ecosystem.
