Technologies: HTML, CSS, JavaScript

Type: Interactive Web Application

Word Doc Synth reimagines a word processor as a playable instrument . letters trigger notes, words form chords, and formatting changes the sound. By combining nostalgic 2000s-style UI with generative music coding, it transforms everyday typing into an expressive, musical experience.

KEY SKILLS

  • JavaScript (Tone.js), HTML, CSS
  • Real-time sound mapping & synthesis
  • UI/UX design and onboarding flow
  • Recursive, modular code structure
  • Interaction design and visual feedback
  • Interactive sound

Inspired by the idea of "writing sound", this project explores how language and interaction can merge into play. The interface intentionally mimics a certain popular word doc editor for instant familiarity, letting users focus on discovering sonic rules rather than learning a new tool. Each text format (bold, italics, underline) manipulates tone in intuitive ways to create a sense of authorship and creation through sound.

I wanted the project to feel playful but grounded in usability: something experimental, yet approachable. Every design decision was made to keep the experience musical, responsive, and fun.
